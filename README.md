# QuicBuild
Building Rust Programs, Made Simple

# Licensing
QuicBuild Falls Under The MCSPSL. Read <a href="https://github.com/MabelMedia-LLC/MCSPSL">The License</a> To Learn More.

# Installation
Just Drop The QuicBuild Binary Anywhere That's In The OS' Executable Path, And Invoke Like So:

QuicBuild [ Name / Folder, cargo/rust Target ] (--debug, --lib)
